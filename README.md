# BrandColors 


## Technologies
Project is created with:
* ReactJs
* SCSS
* React-Virtualized

[Live Demo](https://brandcolors-react.netlify.app/)


## Setup
To run this project, install it locally using npm:

```

 # Run Frontend 
 npm start
 
```
